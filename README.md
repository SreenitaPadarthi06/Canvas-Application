# Interactive Drawing Application

## Features
- Pen Tool
- Eraser Tool
- Line Tool
- Rectangle Tool
- Circle Tool
- Color Picker
- Brush Size Slider
- Undo Functionality
- Clear Canvas
- Save Drawings to Local Storage
- Load Saved Drawings from Gallery
- Export Canvas as PNG
- Touch Support

## Tech Stack
- React
- Vite
- Tailwind CSS
- HTML5 Canvas API

## Installation

```bash
npm install
npm run dev

```

##  Build
```bash
npm run build

```
